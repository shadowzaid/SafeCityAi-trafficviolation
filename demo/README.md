# SafeCityAI Demo

## Demo Flow

1. Double-click `RUN_SAFE_CITY_AI.bat`.
2. Open `http://127.0.0.1:5173`.
3. Visit Dashboard.
4. Run Image Detection.
5. Run Video Detection.
6. Open Live Webcam and capture one frame.
7. Open API Docs and Swagger.
8. Show downloaded annotated image/video.

## Demo GIF Instructions

Use ScreenToGif or OBS:

1. Record the dashboard flow at 1280x720.
2. Export a short GIF as `demo/safecityai-demo.gif`.
3. Export a longer MP4 as `demo/safecityai-demo.mp4`.
